Вкладки                                     ;
Alt + [1-9]                                 ; Перейти к вкладке 1-9
Ctrl + T                                    ; Открыть новую вкладку
Ctrl + W                                    ; Закрыть текущую вкладку

Ctrl + Tab                                  ; Переключиться на предыдущую вкладку
Alt + ⬅️ ➡️                                 ; Назад / Вперед
Ctrl + R                                    ; Перезагрузить вкладку
Ctrl + Shift + R                            ; Жестко перезагрузить страницу (без кэша)
Fn + F11                                    ; Перейти в полноэкранный режим
Ctrl + F                                    ; Найти на странице
Ctrl + G                                    ; Найти на странице следующее
Ctrl + D                                    ; Добавить страницу в закладки

Окна и Пространства;
Ctrl + [1-9]                                ; Перейти к рабочему пространству 1-9
Ctrl + Shift + E                            ; Перейти к следующему рабочему пространству
Ctrl + Shift + Q                            ; Перейти к следующему рабочему пространству
Ctrl + N                                    ; Открыть новое окно
Ctrl + Shift + P                            ; Открыть новое окно в режиме инкогнито
