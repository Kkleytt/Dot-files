Вкладки                             ; 
Ctrl + [1-9]	                    ; Перейти на вкладку [1-9]
Ctrl + T	                        ; Открыть новую вкладку
Ctrl + W	                        ; Закрыть активную вкладку
Ctrl + N	                        ; Открыть новый файл
Ctrl + Shift + T	                ; Отменить закрытие вкладки
Ctrl + Shift + N	                ; Открыть новый файл в области справа
Ctrl + Alt + ➡️	                    ; Перейти вперед
Ctrl + Alt + ⬅️	                    ; Вернуться назад

F2	                                ; Переименовать файл

Ctrl + I	                        ; Сделать выделение - курсивным
Alt + Shift + S	                    ; Создать список
Ctrl + .,	                        ; Добавить свойство файла

Ctrl + K	                        ; Вставить ссылку
Ctrl + D	                        ; Удалить абзац
Ctrl + ➡️	                        ; Начать запись диктофона

Alt + H	                            ; Добавить цветное выделение
